Overview

Code Mill is a plug-in for Code Crusader. Code Mill is invoked by selecting one or more classes in the C++ class tree and then selecting "Create derived class..." on the Tree menu.

First, enter the name of the new class. Then click on the functions that you intend to override. Type in your name as the author and click "Generate." The new source and header files will automatically open in Code Crusader.
